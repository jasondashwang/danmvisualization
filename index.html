Jason Wang Data Visualizations with d3js

These are my first attempts with using the d3js library to visualize different datasets (both static and live data).

They were made in this order (Excuse the ironic limited styling - that was not the focal point of the project):

  • New York University Student Population Bar Graph
  • IMDB Film / TV Ratings Line Graph
  • NYC Graduation Rate Heat Map
  • Random Number Visualizer Animated Line Graph
  • Twitter Live Data Visualization
